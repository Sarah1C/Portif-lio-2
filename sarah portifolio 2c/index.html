Efeito Monalisa: Um Estudo Interativo de Ilusões Visuais

Este projeto utiliza HTML, CSS e JavaScript para criar um efeito visual baseado na famosa pintura "Monalisa" de Leonardo da Vinci, onde os olhos da personagem parecem seguir o observador. Esse fenômeno de ilusão de ótica é reproduzido digitalmente por meio da manipulação de imagens e interatividade no navegador.

O projeto se utiliza de JavaScript para manipular as coordenadas da imagem conforme o movimento do mouse ou da câmera, criando a sensação de que os olhos da pintura estão sempre olhando para o usuário, independentemente da posição em que ele se encontra.

Esse tipo de ilusão visual não é apenas interessante do ponto de vista artístico, mas também oferece uma oportunidade de estudar como o cérebro humano percebe e interpreta imagens estáticas de maneira dinâmica.

Jogo Quente e Frio: Adivinhe o Número!

Este projeto é um jogo interativo baseado no clássico "quente e frio". O objetivo do jogo é adivinhar um número gerado aleatoriamente pelo computador. O jogador recebe dicas se está "quente" (próximo do número) ou "frio" (distante do número), até encontrar a solução.

A interface é simples e acessível, com o uso de JavaScript para gerar o número aleatório e controlar a lógica do jogo. O CSS é utilizado para criar um design minimalista e agradável, enquanto o HTML organiza o conteúdo de forma intuitiva e funcional.

Esse projeto é uma ótima oportunidade para aprender sobre manipulação de números aleatórios e interação com o usuário usando JavaScript.
